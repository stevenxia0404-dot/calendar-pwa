## v0.4.0 重构记录（2026-07-14）

### 决策背景
v0.3.0 卡在部署阶段 3 个月，核心问题复盘：
- **GitHub OAuth** 不适合国内用户（无 GitHub 账号）
- **Coze AI** 增加延迟和成本，90%+ 场景前端规则引擎已覆盖
- **语音输入** 是假的 2 秒动画
- **同步策略** 云端覆盖本地，跨设备编辑会丢数据
- Coze Token 硬编码在源码中，已泄露到 Git 历史

### 改动内容

**前端（src/app/page.tsx）**
- 砍掉 Coze AI 工作流及 `COZE_CONFIG`
- 新增浏览器原生语音识别：按住说话、松开发送、滑动取消（56px 大按钮）
- Safari 不渲染麦克风按钮，用 iOS 键盘自带听写
- GitHub OAuth 替换为邮箱验证码激活（弹窗交互，无感登录）
- 日程 ID 从 Date.now() 换成 crypto.randomUUID()
- 新增 updatedAt 字段，同步策略改为时间戳合并

**后端（workers-api/src/index.js）**
- GitHub OAuth 相关代码全部移除
- 新增邮箱验证码认证：/auth/send-code、/auth/verify、/auth/me、/auth/account
- JWT 修复：btoa 改为标准 base64url 编码，新增 exp 过期
- 新增 MailChannels 免费发邮件
- 同步 API 增强：GET /events?since= 增量拉取、POST /events/merge 合并
- 所有写入支持 INSERT OR REPLACE + updated_at 比较

**配置（workers-api/wrangler.toml）**
- 移除 GitHub OAuth 相关变量
- 新增 SENDER_EMAIL，JWT_SECRET 改为 secret 管理
