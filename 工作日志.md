# 菠萝日程 - 工作日志

## 项目概述
智能日程管理应用，支持自然语言输入、AI解析、本地存储和导出功能。

**技术栈：** Next.js + React + TypeScript + Tailwind CSS + Coze工作流

### v2.0 AI 助理重构（2026-07-19）

#### AI 对话系统
- 移除旧 NLU 输入栏，统一为 AI 对话入口（右下角 💬 浮动按钮，底部滑出 66vh）
- 支持 5 家国内品牌：DeepSeek、Kimi、豆包、通义千问、智谱 GLM
- 模型自由切换（含定价显示），API Key 存浏览器 localStorage 直连 AI 服务商
- 品牌官网、费用明细入口一键跳转，对话框 header 显示当前品牌/模型
- 用户发送消息 → AI 返回确认卡片 → 确认后写入日程/Task
- 系统提示词动态注入当前日期，正确推算"后天""下周X"等
- 30s 超时处理、错误信息展示、Kimi temperature 兼容

#### 文件上传
- `＋` 菜单：📷 照片（调系统相机/相册）、📁 文件、📋 模板
- 拖拽文件到对话框区域直接上传
- PDF 文字提取（pdfjs-dist 动态导入，客户端执行，展示前 5 页）
- XLSX/CSV 智能列映射转文本、图片 base64 缩略图显示
- 文件和图片在聊天框紧凑显示（图标+文件名 / 缩略图）

#### 导入功能改版
- 独立导入弹窗：两步模式（原文展示 → 解析预览），可逐条编辑/删除
- 标准模板 CSV（日期,开始时间,事项,备注），导出格式同步统一
- 导入历史（localStorage 存储，可回溯可删除）
- 支持文件格式：.csv/.xlsx/.xls/.pdf/.txt/.docx/.ics/.json

#### 日历 UI 优化
- 日程和日历左右并排（Desktop：左 1/4 + 右 3/4），填满可用空间
- 日历数字加大加深（text-lg font-medium，#1C1C1C），py-3 铺满
- 底部 padding 缩减（pb-44 → pb-16），内容区更充裕
- Header 手机端自适应（flex-wrap 分两行），⚙ AI 设置入口

#### 安防加固
- CORS 白名单（仅允许 schedule.boluomate.com + 本地开发端口）
- API 限流（D1 rate_limits 表，每 IP 60次/分钟 → 429）
- JWT 过期从 1 年缩短到 7 天
- CSP 双重保护（layout.tsx meta 标签 + public/_headers Cloudflare Pages）
- X-Content-Type-Options、X-Frame-Options、Referrer-Policy 头

#### Bug 修复
- 全量同步从覆盖改为时间戳合并，本地新数据不会被云端旧数据覆盖
- Hydration 错误修复（speechSupported → useState + useEffect）
- + 菜单点击事件修复（遮罩层关闭，不吞 label 选择）
- 导入弹窗关闭时同步关闭 showImportModal

#### 回退
- 小组件模式（半透明磨砂玻璃）→ 无实际场景，已移除
- 旧底部输入栏 + 语音按钮 + NLU handleSubmit → AI 对话入口替换

### v2.1 架构重构（2026-07-21）

#### LLM 函数调用替代 JSON 文本解析
- `sendChat` 改用 OpenAI `tools` 参数，模型直接返回 `tool_calls` 结构体
- 删除手动 JSON 解析/正则清洗/降级 fallback 逻辑
- System prompt 从 12 行精简到 5 行，结构约束由 tools schema 接管
- 单 card → cards 数组，支持一次确认多条日程 + 全部确认按钮

#### 文件解析架构三层分离
- `src/config/fileStrategy.js` — 配置层：文件大小限制 + PDF.js 版本
- `src/utils/fileDispatcher.js` — 调度层：图片 base64 / PDF 文字提取 / 其他透传
- PDF 提取：主线程 `pdfjs-dist` (v6.1.200) + `public/pdf.worker.min.mjs`
  - 硬上限 6000 字，提前终止避免超大 payload
  - 废弃 Web Worker 方案（pdf.js CDN 在 Worker 内访问 document 致命错误）
- 旧 `uploadToChat` 中 FileReader base64 回调 + pdfjs-dist 动态导入 + 5 页 for 循环删除
- 图片 5MB 上限检查

#### 多模型 Token 适配防腐层
- `src/config/modelStrategy.js` — 模型→maxTokens 映射表（对齐 9 个实际模型 ID）
- `src/utils/tokenAdapter.js` — `adaptPayloadToModel` 超限截断 + 控制台警告
- `sendChat` 中 3 处 content 组装点接入防腐层
- 图片消息 `image_url` 通道原样透传，不走 Token 截断

#### 部署修复
- Cloudflare Pages Build System V3 → V2（V3 跳过 `npm run build` 导致部署 404）
- CSP `script-src` 新增 `blob:` 和 `cdnjs.cloudflare.com`
- AI 请求超时 30s → 60s
- 聊天消息区 `min-h-0` 修复 flex 滚动溢出

#### Bug 修复
- PDF worker CDN 版本 4.0.379 → 6.1.200（对齐本地 npm 安装版本）
- 线上部署卡死 2 天：`dist/` 非 `.gitignore`，Cloudflare Pages 直接 serve 静态文件
- Worker 文件 404：从 CDN 改为 `public/` 本地文件
