## 推荐优先级

- **前端项目** → airbnb/javascript + Prettier + ESLint
- **iOS/macOS** → Apple HIG（App Store 审核硬标准）
- **Android** → Material Design 3
- **后端 API** → Google AIP / Microsoft API Guidelines
- **PWA** → Google Workbox + web.dev 官方文档
- **全栈团队** → Kristories/awesome-guidelines（一站式索引）
