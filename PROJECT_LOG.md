# AI日程管家 - 项目开发日志

## 项目概述
智能日程管理应用，支持自然语言输入、AI解析、本地存储和导出功能。

**技术栈：** Next.js + React + TypeScript + Tailwind CSS + Coze工作流

**开发者备注：**
- 混合模式策略：前端优先解析简单日期，复杂语义交由Coze处理
- 错误降级：网络失败时自动使用当前选中日期
- 所有数据存储在浏览器localStorage，无痕模式下数据会丢失
